# homebrew-tap

xinghelee 的 Homebrew tap。

## Berth

Mac 原生 SSH 客户端,跳板机/端口转发/iCloud 同步:[xinghelee/Berth](https://github.com/xinghelee/Berth)

```sh
brew install --cask xinghelee/tap/berth
```

## Termite

macOS 原生终端,会话保活 + 深度 shell 集成:[xinghelee/Termite](https://github.com/xinghelee/Termite)

```sh
brew install --cask xinghelee/tap/termite
```
